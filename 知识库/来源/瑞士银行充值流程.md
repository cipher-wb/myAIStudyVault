# 瑞士银行充值流程

> Source: `.源文档/文字稿/瑞士银行充值流程.md`

## Summary

通过 [[SafePal]] 钱包内置的瑞士银行卡功能，利用 [[OKX]] 交易所 C2C 用支付宝购买 USDC，经提币 → SafePal → 瑞士银行卡的路径完成充值。

## Step-by-step

1. **翻墙设置**：手机上翻墙到**马来西亚**节点（全局模式），确保 OKX 和 SafePal 均可正常访问
2. **OKX 购币**：打开 [OKX](https://www.okx.com/zh-hans) → 点击左上角「C2C 买币」→ 用**支付宝**购买 **USDC**（不推荐 USDT，会多一次转换）
3. **提币到 SafePal**：OKX 右上角「提币」→ 币种选 **USDC** → 网络选 **Arbitrum**（必须）→ 手机端 SafePal 点击 Coin → 收款 → 选择 **USDC（Arbitrum One）** → 复制收款码 → 粘贴到 OKX → 等待约 **2 分钟**到账
4. **转入瑞士银行卡**：打开 SafePal → Bank 页签 → 点击「充值」→ 将 USDC 转为 USD

> [!warning] 瑞士银行卡仅支持 **USDC（Arbitrum One）**，其他币种或网络会失败。
