# 瑞士银行充值流程

> Source: `.源文档/文字稿/瑞士银行充值流程.md`

## Summary

通过 [[SafePal]] 钱包绑定瑞士银行卡，利用 [[OKX]] 交易所用 RMB 购买 USDC，经提币 → SafePal → 瑞士银行卡的路径完成充值。

## Step-by-step

1. **SafePal 设置**：打开 SafePal App → 切换 VPN 到**瑞士** → 出现 Bank 页签
2. **OKX 购币**：在 [[OKX]] 用 RMB 购买 USDC
3. **提币到 SafePal**：OKX → 资产管理 → 提币 → 选择 **USDC（Arbitrum One 网络）**
4. **转入瑞士银行卡**：在 SafePal 中将 USDC 转换到瑞士银行卡

> [!warning] 瑞士银行卡仅支持 **USDC（Arbitrum One）**，其他币种或网络会失败。
