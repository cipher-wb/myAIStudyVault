## Summary

2026-04-24 的学习笔记，覆盖三个主题：

### 1. OpenClaw 备忘单

[[OpenClaw]] 是一个 Bot 网关管理工具。核心操作：
- **启动/关闭/重启**：`gateway start/stop/restart`
- **状态查看**：`gateway status`、`status`（全局）、`logs --follow`
- **交互 UI**：`tui` 或 `terminal`；Web Dashboard 在 `http://127.0.0.1:18789/`
- **通道管理**：`channels list/status/add/remove`，支持 [[QQ Bot]]
- **设备授权**：`devices list/approve --latest`

### 2. API 代充服务

在中国无法直接用信用卡充值 API 服务时，可使用 [[Bewildcard]]（https://bewildcard.com/service）进行代充。已基于此创建了 `gptsapi-image-generator` skill，调用 [[GPT-Image-2]] 进行图像生成。
