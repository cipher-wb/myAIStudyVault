# Anthropic

AI safety 方向的前沿实验室，LLM 厂商之一。创始核心多来自 OpenAI。旗下最知名的产品线是 [[Claude]] 系列模型。

## 关键产品/服务

- **Claude 模型家族**：Opus（旗舰）、Sonnet（平衡）、Haiku（轻量）
- **[[Claude Managed Agents]]**：托管智能体运行时（beta）
- **Console**：控制台与 API key 管理 — [platform.claude.com](https://platform.claude.com/)
- **SDK**：Python (`pip install anthropic`) 等多语言官方库
- **CLI**：[[ant CLI]]（agent/environment 管理）
- **MCP (Model Context Protocol)**：Anthropic 主推的模型-工具通信协议

## 关键资源

- Console / API keys：https://platform.claude.com/settings/keys
- 文档：https://platform.claude.com/docs/
