# GitHub

![[ingest/github-concept.png]]

基于 [[Git]] 的云端托管服务。提供远程仓库、Pull Requests、Issues、Actions 等。

## 关键功能

- **远程仓库**：云端存储代码
- **Pull Requests**：代码审查
- **Issues**：问题追踪
- **Actions**：CI/CD 自动化工作流
