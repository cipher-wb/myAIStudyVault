# Vercel

Next.js 的创建公司。提供针对 Next.js 优化的部署平台。

## 部署优势

- **零配置**：自动识别 Next.js 项目
- **全球 CDN**：快速内容分发
- **免费 Hobby 计划**：新手无需购买服务器
- **GitHub 集成**：push 代码自动部署
- **预览部署**：每个 PR 自动创建预览环境

## 部署流程

1. 注册 Vercel（用 GitHub 账号）
2. 导入 Git 仓库
3. 配置环境变量（如有）
4. 点击 Deploy → 获得 `https://xxx.vercel.app` URL
5. 可绑定自定义域名
