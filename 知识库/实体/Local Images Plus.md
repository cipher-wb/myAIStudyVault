# Local Images Plus

Obsidian 插件。配合 Chrome 的 **Obsidian Web Clipper** 使用，可以把剪藏网页中的外链图片自动下载到本地 `_附件/` 并替换为 `![[]]` 内嵌。

## 使用方式

`Ctrl+P` → 搜 `Local Images` → 对当前笔记或整个 vault 执行一次。
