# Bewildcard

**Bewildcard**（https://bewildcard.com/service）是一个为中国用户提供的 API 服务代理充值平台。解决的核心问题：中国大陆用户无法直接使用信用卡充值海外 API 服务（如 OpenAI API）。

## Use Case

- 通过 Bewildcard 代充后，直接调用 API（如 [[GPT-Image-2]]）
- 已成功用于创建 `gptsapi-image-generator` skill
