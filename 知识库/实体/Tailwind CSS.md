# Tailwind CSS

[[Next.js]] 默认支持的实用优先 CSS 框架。通过类名（如 `bg-blue-500`、`rounded`、`p-4`）快速构建界面，不需要写自定义 CSS。

与 [[shadcn/ui]] 配合使用。
