# shadcn/ui

https://ui.shadcn.com/docs/components

不是传统组件库，而是可以直接复制到项目中的代码。基于 Radix UI + Tailwind CSS。

## 两个用法

1. **开发时**：直接安装组件 → `npx shadcn-ui add button card input`
2. **设计参考**：浏览组件动态效果 → 找到想要的效果 → 向 AI 描述时有了具象参考
