# Git

分布式版本控制系统。追踪文件变化历史，是编程中控制版本的最重要的工具。

## 要点

- **完全离线**工作，不需要网络
- 快速的分支创建和合并
- 强大的历史记录和版本回溯能力
- 与 [[GitHub]] 配合实现远程协作

## 在本库中的使用

这个知识库本身就是 git 仓库，用 [[obsidian-git]] 插件每 30 分钟自动 commit + push 到 GitHub。

详见 [[Git 版本控制]]。
