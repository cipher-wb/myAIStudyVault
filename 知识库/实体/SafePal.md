# SafePal

**SafePal** 是一款加密货币钱包 App，内置瑞士银行卡功能。

## Key Features

- **瑞士银行卡**：App 内 Bank 页签（需翻墙才能显示，推荐马来西亚节点全局模式）
- 支持接收 [[OKX]] 提币的 USDC（Arbitrum One 网络）
- Bank 页签充值：将 USDC 转为 USD
