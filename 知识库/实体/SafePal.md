# SafePal

**SafePal** 是一款加密货币钱包 App，内置瑞士银行卡功能。

## Key Features

- **瑞士银行卡**：App 内 Bank 页签（需 VPN 切换到瑞士才能显示）
- 支持接收 [[OKX]] 提币的 USDC（Arbitrum One 网络）
- 可将 USDC 转换到瑞士银行卡余额
