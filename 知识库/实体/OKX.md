# OKX

**OKX**（https://www.okx.com/）是一个加密货币交易所，支持用法币（如 RMB）购买多种加密货币。

## Key Features

- 支持 RMB 直接购买虚拟货币
- 资产管理 → 提币功能，可转账到外部钱包（如 [[SafePal]]）
- 支持多种网络（包括 Arbitrum One）

## Usage

在瑞士银行充值流程中，OKX 的角色是用 RMB 购买 USDC → 提币到 SafePal。
