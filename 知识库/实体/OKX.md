# OKX

	**OKX**（https://www.okx.com/）是一个加密货币交易所，支持用法币（如 RMB）购买多种加密货币。

## Key Features

- **C2C 买币**：左上角「C2C 买币」入口，支持支付宝购买 USDC（也支持 USDT，但多一次转换不推荐）
- 资产管理 → 提币功能，可转账到外部钱包（如 [[SafePal]]）
- 支持多种网络（包括 Arbitrum One）

## Usage

在瑞士银行充值流程中，OKX 的角色是：C2C 用支付宝购买 USDC → 提币到 SafePal（Arbitrum One，约 2 分钟到账）。
