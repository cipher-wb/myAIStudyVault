# OpenClaw

**OpenClaw** 是一个 Bot 网关管理工具，提供 CLI 和 Web Dashboard 两种管理方式。支持多种聊天通道（如 [[QQ Bot]]），可在本地启动/停止/监控 Bot 服务。

## Key Features

- **CLI 管理**：完整的命令行工具，覆盖网关控制、通道管理、设备授权
- **Web Dashboard**：浏览器访问 `http://127.0.0.1:18789/`
- **TUI 交互**：`openclaw tui` 或 `openclaw terminal` 打开终端交互界面
- **多通道支持**：可添加/删除不同的 Bot 通道（如 QQ Bot）
- **设备授权**：管理连接设备的授权请求
