# 加密货币充值流程

> 通过加密货币中转，将 RMB 充值到瑞士银行卡。
> (Source: [[瑞士银行充值流程]])

## Flow

```
RMB → [OKX 购买 USDC] → USDC（Arbitrum One）→ [提币] → SafePal → [转换] → 瑞士银行卡
```

## Steps

1. **OKX**：用 RMB 购买 USDC
2. **提币**：从 [[OKX]] 提币到 [[SafePal]]，网络选择 **Arbitrum One**
3. **转换**：在 SafePal App 中将 USDC 转入瑞士银行卡

## Critical Notes

> [!warning] 网络必须选择 **Arbitrum One**，选错网络会导致资产丢失。
> [!warning] SafePal 的 Bank 页签需要 VPN 切换到瑞士才能显示。
