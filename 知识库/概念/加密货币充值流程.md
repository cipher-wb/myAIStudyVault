# 加密货币充值流程

> 通过加密货币中转，将 RMB 充值到瑞士银行卡。
> (Source: [[瑞士银行充值流程]])

## Flow

```
RMB → [OKX C2C 支付宝购 USDC] → USDC（Arbitrum One）→ [提币 ~2min] → SafePal → [Bank 充值] → 瑞士银行卡（USD）
```

## Steps

1. **翻墙**：手机翻墙到马来西亚节点（全局模式）
2. **OKX C2C 买币**：支付宝购买 USDC（不推荐 USDT）
3. **提币**：从 [[OKX]] 提币到 [[SafePal]]，网络选择 **Arbitrum One**，约 2 分钟到账
4. **Bank 充值**：SafePal Bank 页签点击充值，USDC 转为 USD

## Critical Notes

> [!warning] 网络必须选择 **Arbitrum One**，选错网络会导致资产丢失。
> [!warning] SafePal 的 Bank 页签需要翻墙才能显示（推荐马来西亚节点，全局模式）。
